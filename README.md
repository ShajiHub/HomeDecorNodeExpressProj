# Home Decor Web Application

A server-side web application built using **Node.js and Express.js** that allows users to browse home décor products through dynamic web pages. This project demonstrates backend web development concepts including routing, middleware, and server-side rendering.

## Technologies Used
- Node.js
- Express.js
- JavaScript
- HTML
- CSS
- Git & GitHub

## Features
- Server-side web application built with Node.js
- Express.js routing to handle multiple pages
- Dynamic rendering of product information
- Structured project architecture for scalability
- Basic frontend interface for product browsing

## Project Structure
```
HomeDecorNodeExpressProj
│
├── routes
├── views
├── public
├── app.js
├── package.json
```

## Installation

1. Clone the repository

```
git clone https://github.com/ShajiHub/HomeDecorNodeExpressProj.git
```

2. Navigate to the project folder

```
cd HomeDecorNodeExpressProj
```

3. Install dependencies

```
npm install
```

4. Run the application

```
node app.js
```

5. Open your browser and visit

```
http://localhost:3000
```
